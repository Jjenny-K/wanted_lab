## 과제 해석
주어진 회사 정보를 포함한 데이터베이스를 설계하여 필터링 및 검색 기능을 제공하는 서비스라고 해석하였습니다.

## 구현 요구사항
-   Database

        -   이름(다국어 지원 가능)
    -   제공되는 DataSet 이용
    -   RDB 사용
-   REST API
    -   [ ] 회사 이름으로 회사 검색기능
        -   회사명 자동완성(이름의 일부만 들어가도 검색 가능)
    -   [ ] 새로운 회사 추가 기능
-   **Implementation**
    -   과제 시작일부터 48시간 이내 구현 종료
    -   유연한 확장과 성능을 고려한 설계 및 구현
    -   [ ] 제공되는 test case 통과 필요
